Why use the DiamondCast App?

Our app helps clients find the help they need to take care of their homes. Clients can easily find contractors and real estate agents that are listed on our app to perform a wide variety of services. Once the client finds and choose their service provider, they can quickly set up an appointment with the provider.

How to use the DiamondCast App

If you have an Android device:

Download and install the latest release on your device. After installation, the application icon will be available for you to access the app.

Otherwise:

Download Android Studio and set up an emulator in the 'AVD Manager'. Download the source code from our latest release and open the 'DiamondCastApp' directory in Android Studio. You can now build and run the app on the Android Emulator.
